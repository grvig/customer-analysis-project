Example Questions:

Question:
How many customers are in the database?

SQL:

Question:
Show the top 5 complaint categories.

SQL:
SELECT issue_category, COUNT(*)
FROM calls
WHERE call_type = 'Complaint'
GROUP BY issue_category
ORDER BY COUNT(*) DESC
LIMIT 5;

Question:
Which branches have the highest customer ratings?

SQL:
SELECT
c.branch,
ROUND(AVG(s.customer_rating), 2)
FROM surveys s
JOIN services sv
ON s.service_id = sv.service_id
JOIN calls c
ON sv.call_id = c.call_id
GROUP BY c.branch
ORDER BY AVG(s.customer_rating) DESC;
